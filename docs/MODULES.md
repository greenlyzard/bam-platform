# BAM Platform — Module Index Pointer

> This used to be the surface/route map. The route map has been renamed
> to `docs/PORTAL_SURFACES.md` (12 portal/route surfaces).
>
> "Module" in this project has two meanings:
>
> 1. **Capability modules (M1–M13)** — domain-level architecture
>    (Communications, Scheduling, Teachers, etc.).
>    Canonical: `docs/strategy/platform-product-requirements.md`
>    Summary: `/CLAUDE.md` Section 7
>
> 2. **Portal/route surfaces** — UI surfaces a user navigates to
>    (Parent Portal, Teacher Portal, Admin Dashboard, etc.).
>    Canonical: `docs/PORTAL_SURFACES.md`
>
> Both views describe the same platform from different angles.
> Pick the one that matches the question you're asking.
